# Ink Colors

Ink tailwind plugin creates a set of component-level class names
that define both text and border colors based on the options
in `theme.colors.ink`.

All `.yobta-ink-*` respect the dark mode setting.

Yobta Ink plugin is useful for creating outlined elements.

## Basic Example

Let's suppose we have an outlined button and we want that button to
have same border and text color.

<Canvas>
  <Story name="Basic Example">
    <div className="flex gap-x-2">
      <button className="yobta-button border-2">Button</button>
      <button className="yobta-button border-2 yobta-ink">Button</button>
    </div>
  </Story>
</Canvas>
