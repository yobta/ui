# Button

Button plugin adds `.yobta-button` class name to the components `@layer`.

## Basic Usage

Default button has transparent background and inherits text color
from it's parent.

Unless you don't need a `busy` state of your buttons it's OK to use an HTML tag with a class name.

<Canvas>
  <Story name="Basic Usage">
    <button className="yobta-button" type="button">
      Button
    </button>
  </Story>
</Canvas>
