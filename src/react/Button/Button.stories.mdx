# Button

Button plugin adds following class names to the components `@layer`
of tailwind styles: `.yobta-list`, `.yobta-list-header`,
`.yobta-list-item`.

To work with list you only need to add those class names to html tags.

## Basic Usage

Default button has transparent background and inherits text color
from it's parent.
