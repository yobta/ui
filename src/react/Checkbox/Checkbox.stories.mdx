# Checkbox

Checkbox plugin adds '.ybta-checkbox' class name to the base @layer of tailwind styles.

## Basic Usage

<Canvas>
  <Story name="Basic Usage">
    <div className="yobta-paper">
      <input type="checkbox" className="yobta-checkbox" />
    </div>
  </Story>
</Canvas>
