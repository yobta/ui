# List

List plugin adds following class names to the components `@layer`
of tailwind styles: `.yobta-list`, `.yobta-list-header`,
`.yobta-list-item`.

To work with list you only need to add those class names to html tags.

## Basic Usage

<Canvas>
  <Story name="Basic Usage">
    <ul className="yobta-list yobta-paper-2">
      <li className="yobta-list-header">List Header</li>
      <li className="yobta-list-item">List Item 1</li>
      <li className="yobta-list-item">List Item 2</li>
      <li className="yobta-list-item">List Item 2</li>
    </ul>
  </Story>
</Canvas>
