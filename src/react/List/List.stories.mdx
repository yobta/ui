# List

List plugin adds following class names to the components `@layer`
of tailwind styles: `.yobta-list`, `.yobta-list-header`,
`.yobta-list-item`.

To work with list you only need to add those class names to html tags.

## Basic Usage

<Canvas>
  <Story name="Basic Usage">
    <section className="yobta-list yobta-paper-2" role="group">
      <h4 className="yobta-list-header">List Header</h4>
      <div className="yobta-list-item" role="listitem">
        List Item 1
      </div>
      <div className="yobta-list-item" role="listitem">
        List Item 2
      </div>
      <div className="yobta-list-item" role="listitem">
        List Item 2
      </div>
    </section>
  </Story>
</Canvas>
