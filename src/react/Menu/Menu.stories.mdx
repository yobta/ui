# Menu

Menu plugin adds following class names to the components `@layer`
of tailwind styles: `.yobta-menu`, `.yobta-menu-header`,
`.yobta-menu-item`.

To create menues you only need to add those class names to html tags.

## Basic Usage

<Canvas>
  <Story name="Basic Usage">
    <menu className="yobta-menu yobta-paper-2">
      <h4 className="yobta-menu-header">Menu Header</h4>
      <button className="yobta-menu-item">Menu Item 1</button>
      <button className="yobta-menu-item">Menu Item 2</button>
      <button className="yobta-menu-item">Menu Item 2</button>
    </menu>
  </Story>
</Canvas>
