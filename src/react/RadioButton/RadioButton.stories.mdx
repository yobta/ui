# Radio Button

RadioButton plugin adds '.yobta-radio' class name to the base @layer of tailwind styles.

## Basic Usage

<Canvas>
  <Story name="Basic Usage">
    <input type="radio" className="yobta-radio" />
  </Story>
</Canvas>
