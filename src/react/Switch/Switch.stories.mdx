# Radio Button

Switch plugin adds '.yobta-switch', '.yobta-switch-small' and '.yobta-switch-big' class names to the base @layer of tailwind styles.

## Basic Usage

<Canvas>
  <Story name="Basic Usage">
    <input type="checkbox" className="yobta-switch" />
  </Story>
</Canvas>
